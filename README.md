# 🔐 Vaultorix

> **A modern, secure file vault for Windows built with Qt and C++.**
>
> Encrypt, decrypt, organize, and manage your sensitive files in one secure desktop application.

## 📖 Overview

**Vaultorix** is a desktop application designed to provide a secure environment for storing confidential files. It combines modern encryption with an intuitive user interface, allowing users to protect their personal and professional data without sacrificing usability.

The application is currently available for **Windows**, with **macOS** and **Linux** support planned for future releases.


## ✨ Features

- Secure account authentication
- Import files into your encrypted vault
- AES-based file encryption
- Restore previously encrypted files
- Open files directly from the vault
- Account management
- Built-in application updater
- Submit feedback from within the application
- More features coming soon

## 🛠️ Technology Stack

- **C++23**
- **Qt 6**
- **CMake**
- **OpenSSL**
- **Qt Widgets**
- **QSql**

## 📋 Requirements

Before building Vaultorix, make sure you have:

- Qt 6 or newer
- CMake 3.12 or newer
- OpenSSL
- Qt Creator *(recommended)* or Visual Studio Code with CMake support


## 🚀 Building the Project
### Clone the repository

```bash
git clone https://github.com/umar-masood/Vaultorix.git
cd Vaultorix
```

### Configure

```bash
cmake -B build
```

### Build

```bash
cmake --build build
```

Alternatively, you can simply open the project in **Qt Creator** and build it from there.
